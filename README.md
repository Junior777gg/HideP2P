Быстрый старт (Пример использования)
Вся работа с библиотекой сводится к простым шагам: создать менеджер, обменяться данными, соединиться и начать слать байты.
Важные ограничения (Что нужно знать)
Мобильный интернет и Symmetric NAT:
Данная библиотека использует технологию UDP Hole Punching. Эта технология физически не работает, если оба устройства находятся за жестким мобильным Symmetric NAT (CGNAT) с агрессивной сменой портов.
