# HideP2P
Библиотека для работы по Peer-to-Peer
## Быстрый старт (Пример использования)
Вся работа с библиотекой сводится к простым шагам:
1. Создать менеджер
2. Обменяться данными
3. Соединить пользователей
4. Начать отсылать байты
## Важные ограничения (Что нужно знать)
### Мобильный интернет и Symmetric NAT:
> [!WARNING]  
> Данная библиотека использует UDP Hole Punching. Эта технология физически не работает, если оба устройства находятся за жестким мобильным Symmetric NAT (CGNAT) с агрессивной сменой портов.
